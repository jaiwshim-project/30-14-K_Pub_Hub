Quick Start

빠른 시작 가이드

3단계만 거치면 바로 교육을 시작할 수 있습니다.

1

교육 세션 생성

관리자가 회사명, 교육일, 비밀번호를 설정하여 새로운 교육 세션을 생성합니다.

2

교육생 등록

팀을 편성하고 교육생의 이름과 이메일을 입력하여 등록합니다.

3

교육 시작

강사 또는 교육생이 회사 선택 → 비밀번호 입력 → 역할 선택 후 교육을 시작합니다.

Features

12가지 핵심 기능

SPIN Selling 교육에 필요한 모든 기능을 제공합니다.

📖

프리리딩

SPIN 2.0 사전학습 PDF (28페이지). 교육 전 핵심 개념을 미리 학습합니다.

✓

SPIN 개념 퀴즈

핵심개념, 구매사이클, SPIN 유형, 고급 문제 등 총 27문제로 이해도를 검증합니다.

🔍

니즈 구분

잠재니즈(Implied Needs) vs 현재니즈(Explicit Needs) 구분 실습을 진행합니다.

💬

SPIN 질문구분

영업 대화에서 S(상황)/P(문제)/I(시사)/N(해결) 질문 유형을 분류합니다.

✎

질문 작성 연습

12개 산업분야별 SPIN 질문을 직접 작성하고 AI가 실시간 평가합니다.

🎯

SPIN질문&니즈 구분

비브라디페랑스 시나리오 기반 대화형 퀴즈로 질문과 니즈를 통합 실습합니다.

💎

FAB/BAF 연습

Feature(특징), Advantage(장점), Benefit(이점)을 구분하고 활용하는 연습입니다.

🤖

AI 롤플레이

10개 산업분야, 3단계 난이도로 AI 고객과 실전 상담 시뮬레이션을 진행합니다.

📋

콜플랜 작성

6개 섹션의 체계적 상담 계획을 작성하고 AI가 분석 평가합니다.

🔧

스핀 도구들

사례분석 도구, 콜플랜 도구, 스코어보드, 코칭리포트 등 다양한 도구를 제공합니다.

📈

AI 분석리포트

개인별 9+1섹션 AI 역량 진단 리포트를 생성하고 이메일로 발송합니다.

📊

사전/사후 설문조사

교육 전후 자기평가와 만족도 조사를 실시하여 교육 효과를 측정합니다.

상세 기능 설명

플랫폼의 핵심 구조와 평가 체계를 확인하세요.

👥 이해관계자

▼
  • 시스템 관리자 - 교육 세션 생성, 교육생 등록, 전체 데이터 관리
  • 강사 - 교육 진행, 스코어보드 확인, 코칭리포트 작성, AI 분석리포트 발송
  • 교육 매니저 - 교육 일정 관리, 교육생 출석 확인, 결과 리포트 조회
  • 교육생 - SPIN 학습 활동 수행, 퀴즈/연습/롤플레이 참여, 리포트 수신

🏆 점수 체계

▼
  • 문제당 배점 - 10점 기준
  • 활동별 만점 - 80점 ~ 120점 (활동 유형에 따라 상이)
  • 총 만점 - 600점
  • 퀴즈, 니즈 구분, 질문구분, 질문 작성, FAB/BAF, 롤플레이, 콜플랜 등 각 활동의 점수가 합산됩니다.

🧠 AI 평가 기준

▼
  • SPIN 균형 - S/P/I/N 질문의 적절한 비율 유지 여부
  • 질문 순서 - SPIN 순서에 맞는 자연스러운 질문 흐름
  • 상황질문 비율 - 과도한 상황질문(S) 사용을 제한
  • 시사질문 활용 - 문제의 심각성을 부각하는 시사질문(I)의 효과적 활용
  • 해결질문 - 고객이 스스로 해결책의 가치를 인식하도록 유도하는 해결질문(N) 품질

🔬 온톨로지 분석

▼

AI가 교육생의 학습 데이터를 체계적으로 분석합니다:

  • 약점 파악 - 학습 데이터에서 부족한 영역 식별
  • 이슈 도출 - 약점으로 인한 실무 영향 분석
  • 근본원인 추적 - 이슈의 근본적인 원인 규명
  • 개선전략 수립 - 맞춤형 개선 전략 제시
  • 실행과제 제안 - 구체적인 실행 항목과 우선순위 제시

📅 30일 실행계획

▼
  • 1주차: 개념복습 - SPIN 핵심 개념 재정리, 프리리딩 자료 복습
  • 2주차: 집중훈련 - 취약 영역 집중 연습, 질문 작성 반복 훈련
  • 3주차: 실전적용 - 실제 고객 상담에 SPIN 기법 적용, 콜플랜 실행
  • 4주차: 자기점검 - 실행 결과 자기평가, 개선점 도출, 지속 학습 계획 수립

📧 이메일 발송

▼
  • 발송 방식 - Supabase Edge Function + Resend API를 통한 자동 발송
  • 발송 내용 - AI 분석리포트, 30일 실행계획, 온톨로지 분석 결과
  • 수신 대상 - 교육생 개인 이메일 (등록 시 입력한 주소)
  • 발송 시점 - 강사가 AI 분석리포트 생성 완료 후 수동 발송

FAQ

자주 묻는 질문

궁금한 점을 빠르게 확인하세요.

교육생은 어떻게 접속하나요?
+
메인 페이지에서 회사(교육 세션)를 선택하고, 관리자가 설정한 비밀번호를 입력한 뒤, "교육생" 역할을 선택하면 바로 교육을 시작할 수 있습니다. 별도의 회원가입이 필요하지 않습니다.
점수는 어떻게 계산되나요?
+
문제당 10점을 기준으로 각 활동별 만점(80~120점)이 다릅니다. 퀴즈, 니즈 구분, 질문구분, 질문 작성, FAB/BAF, AI 롤플레이, 콜플랜 등 모든 활동 점수가 합산되어 총 600점 만점으로 평가됩니다.
AI 분석리포트는 어디서 볼 수 있나요?
+
강사가 AI 분석리포트를 생성하면, 교육생이 등록한 이메일로 자동 발송됩니다. 또한 플랫폼 내 "내 리포트" 메뉴에서도 직접 확인할 수 있습니다. 리포트에는 9+1섹션의 역량 진단, 온톨로지 분석, 30일 실행계획이 포함됩니다.
강사와 교육생의 차이점은?
+
강사는 전체 교육생의 스코어보드를 확인하고, 코칭리포트를 작성하며, AI 분석리포트를 생성 및 이메일 발송할 수 있습니다. 교육생은 자신의 학습 활동(퀴즈, 연습, 롤플레이 등)을 수행하고 개인 점수와 리포트를 확인할 수 있습니다.
오프라인에서도 사용 가능한가요?
+
현재 버전은 온라인 환경이 필요합니다. AI 평가, 데이터 저장, 이메일 발송 등 핵심 기능이 Supabase 서버와 연동되어 있어 인터넷 연결이 필수입니다. 프리리딩 PDF는 사전에 다운로드하여 오프라인에서 열람할 수 있습니다.
교육생 수 제한이 있나요?
+
하나의 교육 세션당 최대 35명(5팀 x 7명)까지 등록할 수 있습니다. 더 많은 인원이 필요한 경우 별도의 교육 세션을 추가로 생성하여 운영할 수 있습니다.
이메일 발송은 어떻게 하나요?
+
강사 화면에서 AI 분석리포트를 생성한 후, "이메일 발송" 버튼을 클릭하면 Supabase Edge Function과 Resend API를 통해 교육생의 등록 이메일로 자동 발송됩니다. 발송 상태는 실시간으로 확인할 수 있습니다.
교육 데이터는 어디에 저장되나요?
+
모든 교육 데이터는 Supabase의 PostgreSQL 데이터베이스에 안전하게 저장됩니다. 교육 세션 정보, 교육생 답변, 점수, AI 분석 결과 등이 체계적으로 관리되며, 교육 종료 후에도 데이터를 조회할 수 있습니다.
