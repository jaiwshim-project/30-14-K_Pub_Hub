1.저작물 개요

본 명세서는 SPIN Selling AI 교육 플랫폼의 전체 시스템 구조를 시각화한 SVG 다이어그램 architecture.svg에 대한 미술저작물 등록용 기술 명세서이다. 해당 저작물은 20년 이상의 SPIN Selling 교육 경험을 바탕으로 설계된 복잡한 교육 플랫폼의 3계층 아키텍처와 6개 기능 그룹을 한눈에 파악할 수 있도록 독창적으로 시각화한 도형저작물이다.

2.창작 의도 및 배경

2.1 창작 배경

저작자는 Huthwaite International의 공식 파트너이자 Korn Ferry의 SPIN Selling 교육 전문 컨설턴트로서 20년 이상의 기업 영업교육 경험을 축적해 왔다. 이 과정에서 전통적인 오프라인 교육이 가지는 한계 — 반복 학습의 어려움, 개별 피드백의 부족, 학습 성과 측정의 한계 — 를 극복하기 위해 AI 기반의 교육 플랫폼을 직접 기획·설계하였다.

이 플랫폼은 60개 이상의 HTML 페이지, 학습 게임 5종, AI 챗봇 2종(Gemini/OpenAI), Supabase 기반 데이터 계층을 포함하는 복합 시스템으로, 그 구조가 매우 복잡하다. 교육 담당자·강사·학습자·개발자 등 다양한 이해관계자가 이 시스템의 전체상을 한눈에 파악할 수 있도록 시각화한 결과물이 본 저작물이다.

2.2 기존 다이어그램과의 차별점

  • 6개 기능 그룹으로 확장 — 일반적인 3계층(클라이언트/서버/데이터) 구조를 넘어, 학습 게임·AI 통합·부가 기능을 별도 그룹으로 분리하여 시각화
  • Korn Ferry 브랜드 색상 체계 — 네이비 #0a2240 기반 헤더 + 골드 #c9a961 액센트로 브랜드 일관성 확보
  • 계층 간 데이터 흐름 시각화 — 점선 곡선 화살표로 정보 전달 경로 표현
  • 50+ 컴포넌트의 체계적 배치 — 카테고리별 색상 코딩으로 직관적 이해 지원
  • 한국어 레이블 + 이모지 병기 — 국내 사용자의 가독성 극대화
창작 목적
본 다이어그램은 단순한 기술 문서가 아니라, SPIN Selling 교육 철학을 시각적으로 표현한 저작자 고유의 디자인 언어이다. 복잡한 시스템을 아름답고 명료하게 전달하는 것 자체를 하나의 미적 목표로 삼아 제작되었다.

4.2 캔버스 레이아웃 (viewBox 1200 × 1420)

저작물은 수직 확장형 레이아웃으로, 6개의 기능 영역이 위에서 아래로 배치된다. 각 영역은 고유한 색상 헤더와 둥근 모서리 박스(rx 8~12)로 구분되며, 드롭쉐도우 필터로 입체감을 부여하였다.

1클라이언트 계층

y: 60 ~ 280
4가지 역할(관리자·강사·매니저·학습자)별 60+ HTML 페이지 분류. PWA 지원.

2서버 계층

y: 320 ~ 520
Supabase REST API, Realtime, Edge Functions, 인증/권한 모듈.

3데이터 계층

y: 560 ~ 700
PostgreSQL 3개 테이블 (sessions, participants, reports). Storage.

4외부 서비스

y: 740 ~ 880
6개 외부 API 연동 — Gemini, OpenAI, Supabase, Vercel, Analytics 등.

5학습 게임 시스템

y: 930 ~ 1130
5종 게임(Flash·Match·Order·Quiz Race·Industry) + 게임 허브 + 리더보드.

6AI 챗봇 + 부가 기능

y: 1160 ~ 1380
SPIN AI Bot + 12개 부가 기능 페이지(로드맵·FAB·매뉴얼 등).

4.3 디자인 요소

  • 둥근 모서리 박스 — rx 값 8~12로 부드러운 인상 연출
  • 그라데이션 헤더 — 각 그룹별 고유 색상의 좌→우 그라데이션
  • 드롭쉐도우 필터 — feDropShadow로 깊이감 부여 (opacity 0.08~0.12)
  • 점선 화살표 — stroke-dasharray로 데이터 흐름 표현
  • 한국어 레이블 — Segoe UI 폰트로 한글·영문 혼용 가독성 확보
  • 이모지 병기 — 핵심 기능에 이모지를 배치하여 직관성 향상

5.창작성 인정 요소

본 저작물은 단순한 기술 다이어그램을 넘어, 아래 7가지 측면에서 저작자의 독창적 표현이 구현된 미술저작물로서 창작성을 인정받을 수 있다.

  1. 6그룹 확장 구조 일반적인 3계층(클라이언트·서버·DB) 다이어그램의 한계를 극복하고, 학습 게임·AI 통합·부가 기능을 별도 그룹으로 분리한 독자적 분류 체계. 교육 플랫폼의 특성을 반영한 저작자 고유의 설계.
  2. Korn Ferry 브랜드 일관성 네이비 #0a2240 + 골드 #c9a961 의 2색 브랜드 체계를 헤더·타이틀·액센트에 일관되게 적용하여, 플랫폼 전체의 시각적 정체성을 구축.
  3. 카테고리 색상 코딩 6개 기능 그룹을 각각 Blue / Green / Amber / Violet 계열로 구분하고, 같은 그룹 내 하위 요소는 명도 단계로 세분화하여 한눈에 구조를 파악할 수 있게 함.
  4. 계층 간 데이터 흐름 시각화 점선 곡선 화살표(marker + stroke-dasharray 조합)로 클라이언트→서버→데이터 흐름과 외부 API 호출 방향을 명료하게 표현.
  5. 이모지 + 한국어 레이블의 직관적 결합 핵심 기능에 이모지를 앞세우고 한국어 명칭을 병기하여, 기술 문서임에도 학습자·강사·관리자 모두가 직관적으로 이해 가능하도록 설계.
  6. 컴포넌트 카드 디자인 50개 이상의 페이지·기능을 개별 카드로 표현하고, 카드별 배경·테두리·그림자를 통일된 스타일로 적용하여 시각적 리듬감을 형성.
  7. 수직 확장 레이아웃 viewBox 1420 높이의 세로 긴 캔버스를 채택하여, 모든 컴포넌트를 스크롤 없이 한 화면에 담을 수 있도록 구성. A4 인쇄 및 웹 임베드 양쪽에 최적화.

6.발전 과정 (v1.0 → v2.0)

본 저작물은 2025년에 최초 제작된 v1.0을 기반으로, 플랫폼 기능 확장에 따라 2026년 4월에 v2.0으로 전면 개정되었다. 두 버전의 주요 차이점은 다음과 같다.

v1.0 (2025년)
  • viewBox 1200 × 920
  • 25개 HTML 페이지
  • 3계층 단순 구조
  • 기본 3색 팔레트
  • 학습 게임 미포함
  • AI 챗봇 미통합
  • 단일 헤더 디자인
v2.0 (2026년 4월)
  • viewBox 1200 × 1420
  • 50+ HTML 페이지
  • 6그룹 확장 구조
  • 5색 + KF 브랜드 팔레트
  • 5종 학습 게임 추가
  • SPIN AI Bot 통합
  • 그룹별 그라데이션 헤더

6.1 v2.0 주요 개선 사항

  • 신규 그룹 추가 — 학습 게임 시스템, AI 챗봇 통합, 부가 기능 3개 그룹 신설
  • 캔버스 확장 — 세로 높이를 920에서 1420으로 확장 (약 54% 증가)
  • 브랜드 체계 도입 — Korn Ferry Navy + Gold 색상 체계 전면 적용
  • 그라데이션 헤더 — 단색 헤더에서 그라데이션 헤더로 업그레이드
  • 드롭쉐도우 필터 — feDropShadow 2종(shadow, shadowSm) 도입
  • 화살표 마커 — 4색 arrow marker로 데이터 흐름 방향 시각화

8.정보 구조 및 활용 용도

8.1 시각화된 정보 구조

본 저작물은 아래 50개 이상의 컴포넌트를 체계적으로 시각화하고 있다.

클라이언트 계층60+ HTML 페이지 (관리자·강사·매니저·학습자 4개 역할별 분류)
서버 계층Supabase REST API + Realtime + Edge Functions + Auth
데이터 계층3개 PostgreSQL 테이블 (sessions, participants, reports) + Storage
외부 서비스6개 외부 API — Gemini, OpenAI, Supabase, Vercel, GA4, Sentry
학습 게임 시스템5종 게임(Flash·Match·Order·Quiz Race·Industry) + 허브 + 리더보드
AI 통합SPIN AI Bot + Gemini API + OpenAI API (프리미엄 티어)
부가 기능12개 페이지 (로드맵·FAB·매뉴얼·온톨로지·경쟁사 분석 등)

  • 매뉴얼 임베드 — SPIN Selling AI 교육 플랫폼의 매뉴얼 페이지 (manual.html)에 시스템 구조 이해용 공식 도식으로 임베드
