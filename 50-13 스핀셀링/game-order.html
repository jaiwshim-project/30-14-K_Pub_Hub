🔢 SPIN 순서 정렬

한 산업의 4개 질문을 S → P → I → N 순서로 정렬하세요. 클릭한 순서대로 번호가 매겨집니다.

🔄 무한 라운드 ⏱ 시간 제한 없음 🎴 라운드당 4질문 🏆 정답 라운드 +50
📖 게임 가이드 — 클릭하여 펼치기/접기
🎯 목적
SPIN의 핵심인 질문 흐름(S → P → I → N) 순서를 체득합니다. 영업 상담에서 자연스러운 SPIN 흐름을 구사하려면 이 순서가 몸에 배어야 합니다.
🏆 목표
한 산업의 4개 질문을 올바른 SPIN 순서로 정렬합니다. 최대한 많은 라운드를 한 번에 정답으로 통과하세요.
📝 참가 방법
  1. 화면 상단의 산업명을 확인합니다
  2. 4개의 무작위 순서 질문 카드를 봅니다
  3. S(상황) → P(문제) → I(시사) → N(해결) 순서대로 클릭
  4. 1, 2, 3, 4 번호가 자동으로 매겨집니다
  5. "정답 확인" 버튼으로 검증
⚔️ 대결 방식 & 점수
  • 개인 도전 — 무한 라운드, 본인 한계까지 도전
  • 정답 라운드: +50점 누적
  • 오답 시 틀린 위치에 흔들림 효과 표시
  • "다시 선택" 버튼으로 재시도 가능
  • 10라운드 정답 = 500점, 20라운드 = 1000점
