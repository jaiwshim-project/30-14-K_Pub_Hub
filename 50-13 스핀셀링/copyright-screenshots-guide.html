1. 캡처 사양 및 제출 규칙

한국저작권위원회는 컴퓨터프로그램저작물 등록 시 실제 구동 화면을 증빙 자료로 함께 제출할 것을 권장합니다. 본 플랫폼의 30개 핵심 화면을 다음 사양에 맞춰 캡처·제출합니다.

[1-1] 캡처 권장 사양

해상도1920 × 1080 (Full HD) 1순위 · 1440 × 900 2순위 — 브라우저 전체 창 또는 document.body 전체 길이
파일 형식PNG (무손실, 투명 배경 없음). JPG는 텍스트 번짐으로 비권장
컬러 모드sRGB 24bit. 다크모드 사용 시 라이트 테마로 통일 권장
브라우저Chrome 최신 버전 (배율 100%). 주소창/북마크바 숨김 상태 권장
캡처 도구 ① Chrome DevTools → "Capture full size screenshot" (F12 → Ctrl+Shift+P)
② puppeteer-core (이 프로젝트에 글로벌 설치됨)
③ PowerShell + System.Drawing (CLAUDE.md 스크립트)
해상도 예외수료증(30번) 및 팀 대항전(28번)은 가로 방향/대형 스크린 화면이므로 가로 1920 기준

[1-2] 파일명 규칙

형식: spin-screenshot-NN.png (NN은 01~30)
예시: spin-screenshot-01.png → 랜딩 페이지
       spin-screenshot-02.png → 비밀번호 입력 모달
       ...
       spin-screenshot-30.png → 수료증

[1-3] 개인정보 마스킹 지침

저작권 등록 자료는 위원회 내부 자료로 보존되지만, 교육생 실명·이메일·회사 기밀은 반드시 마스킹한 상태로 캡처합니다.
· 실명은 "홍길동", "김철수" 등 더미 이름으로 대체
· 회사명은 "A사", "샘플기업" 등으로 대체
· 이메일은 test@example.com 사용
· 점수·팀 편성 등 학습 데이터는 실제 더미 데이터로 채워진 상태에서 캡처하여 기능이 보이도록 합니다

[1-4] 화면 캡처 제출 구성

총 30장을 spin-screenshots.zip 하나의 압축 파일로 제출합니다.
각 캡처는 해당 페이지에서 가장 대표적인 상태(교육 중 일반적인 사용 화면)로 캡처합니다.
모달이 포함된 화면(2번)은 모달이 열린 상태로 캡처합니다.

2. 화면 캡처 목록 (30장)

SPIN Selling AI 교육 플랫폼의 핵심 화면 30개를 학습 흐름 순서로 정리했습니다. 각 화면은 저작물의 창작적 표현(UI 구성·교육 방법론·AI 기능)을 대표합니다.

번호 화면명 파일 (URL) 핵심 기능 / 캡션 캡처 권장 사항
A. 진입 · 인증 플로우 (1 ~ 3)
01 랜딩 페이지 index.html 히어로 이미지 + 진행 중 교육 세션 카드 목록. 교육생이 자신이 참여할 회사명을 선택하는 시작 화면. FHD 전체, 세션 2~3개 등록된 상태
02 비밀번호 입력 모달 index.html (pwModal) 세션 선택 후 4자리 숫자 비밀번호를 입력하는 모달. 회사별 입장 제어로 데이터 분리. 모달 열린 상태로 캡처
03 역할 선택 화면 company.html 강사/교육매니저/교육생 중 역할 선택 + 팀(A~E) + 이름(슬롯) 선택. 역할에 따라 권한이 분기됨. 3개 역할 카드 모두 보이게
B. 대시보드 및 관리자 (4 ~ 7)
04 교육생 대시보드 company.html (trainee) 12개 학습 활동 카드(프리리딩·퀴즈·니즈·SPIN·질문연습·FAB·롤플레이·콜플랜 등) + 본인 진행률/점수 바. 12개 카드가 모두 보이는 세로 스크롤 포함
05 강사 대시보드 company.html (trainer) 강사 전용 관리 도구(스코어보드/코칭리포트/AI리포트/수료증 등) + 팀별 평균 진행률 요약. 관리 툴바가 포함되도록 FHD
06 시스템 관리자 페이지 admin.html 세션 생성/삭제/종료 토글, 교육생 명단 등록, 비밀번호 설정, LMS 내보내기 등 마스터 권한 도구. 관리자 패널 펼친 상태
07 팀 편성 페이지 team-setup.html 5개 팀 × 7명 슬롯에 교육생 이름을 드래그·입력하여 팀을 편성. 팀명 변경 및 일괄 저장. 5개 팀 카드 가로 배열 전체
C. 핵심 학습 플로우 (8 ~ 15)
08 프리리딩 (PDF 뷰어) prereading.html SPIN 3.0 ILT 한글 교재 PDF 인라인 뷰어. 로그인 불필요, 교육 전 사전 학습용. PDF 1페이지가 보이는 상태
09 퀴즈 진행 화면 quiz.html SPIN 개념·구매사이클·질문유형 4개 카테고리, 총 80점 만점 객관식 퀴즈. 진행률 바 + 점수 누적. 문제 1개 풀이 중 상태
10 니즈 구분 needs.html 잠재니즈(Implied)/현재니즈(Explicit) 식별 연습. 고객 발언을 카드로 분류. 만점 80점. 카드 드래그 드롭 영역 포함
11 SPIN 질문 분류 scenario.html 시나리오 속 질문이 S/P/I/N 중 무엇인지 분류. 산업별 상황 + 객관식 4지선다. 만점 100점. 시나리오 지문 + 선택지 노출
12 질문 연습 practice.html 11개 산업(IT/제조/금융/의료 등) 시나리오로 SPIN 질문을 직접 작성. AI가 적절성 평가 후 점수. 산업 선택 → 질문 작성 화면
13 FAB/BAF fab.html Feature(특징)-Advantage(장점)-Benefit(이점) 프레임워크 연습. 제품을 고객 이점으로 변환. 3단 구조 입력 폼 전체
14 롤플레이 산업 선택 roleplay.html 10개 산업 × 3단계 난이도 시나리오 선택. Gemini AI가 고객 역할을 연기하는 실전 대화 훈련. 산업 카드 그리드 전체
15 콜플랜 작성 callplan.html 6개 섹션 콜플랜(상담목적/배경/SPIN/FAB/반론/Advance) 작성 + Gemini AI 자동 평가 리포트. 6개 섹션 모두 입력된 상태
D. 도구 · 리포트 · 매뉴얼 (16 ~ 22)
16 스코어보드 scoreboard.html 5개 팀 × 7명 점수 매트릭스. 6개 카테고리별 점수 + 팀 합계 + 개인 총점. 실시간 갱신. FHD, 전체 매트릭스 표시
17 코칭 리포트 coaching.html 강사가 교육생 개별로 작성하는 관찰 코멘트 + 강점/약점/액션 플랜 리포트. 저장 및 인쇄 가능. 리포트 본문 전체
18 AI 분석리포트 my-report.html Gemini AI가 개인 점수·약점을 분석해 생성하는 맞춤 역량 진단서. 이메일 발송 기능 포함. AI 생성 리포트 본문 전체
19 온톨로지 그래프 ontology.html 약점 → 이슈 → 근본원인 → 전략 → 실행과제 5단계 역량 진단 그래프(SVG 기반). 그래프 전체 + 범례
20 교육 매뉴얼 manual.html 35개 기능 카드 + 사용법 + FAQ. 프리미엄 화이트 테마로 제작된 플랫폼 공식 매뉴얼. 35개 카드 중 상단 2~3행
21 SPIN 질문 라이브러리 spin-library.html 11개 산업 × 132개 우수 질문 DB. 산업·유형·키워드 필터. 각 질문에 활용팁 포함. 필터 UI + 질문 카드 5~6개
22 교육 자료실 materials.html 14개 교육 자료(교재·템플릿·케이스·참고) 4개 카테고리 분류. PDF/이미지 프리뷰. 자료 카드 그리드 전체
E. 게임 허브 (23 ~ 28)
23 게임 허브 games-hub.html 5종 학습 게임 진입 페이지 — 퀴즈레이스·매칭·플래시카드·순서정렬·다음질문선택. 5개 게임 카드 전체
24 퀴즈 레이스 game-quiz-race.html 30초/문제 타이머 + 10문제 + 콤보 보너스 4지선다 레이스 게임. 최대 약 145점. 타이머·점수·문제 모두 보이게
25 매칭 게임 game-match.html 60초 내 질문 4개 ↔ SPIN 유형 4개 매칭. 라운드 클리어 보너스. 무한 라운드. 매칭 카드 8개 배치 상태
26 플래시카드 game-flash.html 132개 SPIN 질문 카드 무작위 출제. 질문을 S/P/I/N 중 선택. 정답 설명 + 활용팁 제공. 질문 카드 + 4개 선택지
27 게임 리더보드 game-leaderboard.html 5개 게임별 Top 10 + 팀별 합산 점수. 6개 탭 + 5초 자동 갱신 + 1~3위 메달. 전체 탭 + 리더보드 표
28 팀 대항전 모드 team-battle.html 대형 스크린용 실시간 팀 점수 막대 그래프. 1위 팀 골드 글로우. 전체화면 모드 지원. 전체화면 모드로 FHD 캡처
F. AI 기능 · 수료 (29 ~ 30)
29 SPIN AI Bot 챗봇 chatbot.js (floating) 모든 페이지 우하단 플로팅 챗봇. Gemini 2.0 Flash + RAG 지식베이스 + 개인화 코칭 기능. 챗봇 창 열린 상태, 대화 2~3턴
